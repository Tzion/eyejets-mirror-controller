Vector Pattern Memory

Vector Pattern Memory system ID: 0x26

Vector Pattern Memory system is a memory that is shared by all Vector Pattern Unit systems of all channels.

Register Map:

Address Name Default Value Description Type Access
0x2600 Vector pattern data valid false Writing 0 to this register causes the firmware to write the vector pattern memory contents (RAM) to the flash memory boolean read write

Vector Map:

Address Name Default Value Description Type Access
0x2600 Vector pattern data N/A Contents of the Vector Pattern Memory.
Only addressable by units of uint32 (all indices and sizes must be divisible by 4) 		uint32[3000] read write
Attention
Vectors have their own address space.
