Board EEPROM

Board EEPROM system ID: 0x20

Board EEPROM system provides read and write functionality to the EEPROM located in the driver. By default EEPROM data are write protected. To write data, the user has to explicitly unlock the EEPROM by writing the key value to the lock register.

Register Map:

Address Name Default Value Description Type Access
0x2000 Board EEPROM lock 1 (locked) Write the key to unlock, anything else to lock.
Key value: 0x3f4744f6 		uint32 read write

Vector Map:

Address Name Default Value Description Type Access
0x2000 Board EEPROM data N/A Supports bytes up to 0x7F uint8[] read write
0x2001 Board serial number N/A 8 bytes long ascii string read only
Attention
Vectors have their own address space.
