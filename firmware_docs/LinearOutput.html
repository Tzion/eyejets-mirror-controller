MR-E-2  138-161-00
Linear Output

Linear output system base ID: 0xE8

Register Map:

Address Name Default Value Description Type Access
0xE[8-F]00 Current limit 0.7 Limit in amperes where the output clamps float read write
0xE[8-F]01 Channel number N/A uint32 read only
0xE[8-F]02 Output current N/A Current current output in amperes float read only
0xE[8-F]03 Average limit period 0.5 Number of seconds to average current float read write
0xE[8-F]04 Average limit sum 0.5 Maximum allowed average current in amperes float read write
0xE[8-F]05 Cool-down period 10.0 Length of cool-down period in seconds float read write
Note
Full register address varies according to the channel.

For example:

Current limit register of channel one has address 0xE800

Output current register of channel two has address 0xE902
