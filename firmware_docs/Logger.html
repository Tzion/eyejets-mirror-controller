MR-E-2  138-161-00
Logger

Logger system ID: 0x24

A system for logging the values of various readable registers.

Configure the logger by setting the addresses of registers to log in registers 0x2400 to 0x2405. Choose the desired sampling rate and set the run register to start logging. The logged values will be saved in the corresponding vectors 0x2400 to 0x2405. Each vector can save up to 5000 values. This memory is statically allocated to each vector. At default maximum sampling rate of 10kHz this will result in 0.5 seconds of logging time. To increase the logging time, the sampling rate of the logger has to be reduced.
