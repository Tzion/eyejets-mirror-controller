Analog Input

Analog Input system base ID: 0x58

Once this system is activated, the input to the Signal Flow is determined by the analog input signals available to the driver. The analog inputs are sampled at controller frequency which is 10kHz. Depending on the active control mode configured by the Signal Flow Manager the maximum analog input range is mapped linearly to the maximum range of the corresponding unit set by the control mode.

Register Map:

Address Name Default Value Description Type Access
0x5[8-F]00 ADC raw value N/A Converted ADC raw value uint16 read only
0x5[8-F]01 Active input type N/A Determined by active control stage system Unit Type read only
Note
Full register address varies according to the channel.

For example:

ADC raw value register for channel two has address 0x5900

Active input type register for channel one has address 0x5801
