MR-E-2  138-161-00
Temperature Manager

Temperature Manager system ID: 0x22

The temperature manager system provides information about the actuator and the driver temperatures. Temperature limits can be set explicitly to prevent overheating.

Register Map:

Address Name Default Description Type Access
0x2200 Actuator temperature N/A Degrees Celsius uint8 read only
0x2201 Proxy board temperature N/A Degrees Celsius uint8 read only
0x2202 Actuator temperature threshold 85 Degrees Celsius. If this threshold is reached the output of the driver is disabled to cool down. Temperature has to fall by 10 degrees so that the output is enabled again uint8 read write
0x2203 Proxy board temperature threshold 85 Degrees Celsius. If this threshold is reached the output of the driver is disabled to cool down. Temperature has to fall by 10 degrees so that the output is enabled again uint8 read write
