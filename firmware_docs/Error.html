Error Codes

The error code support

System-wide error codes.

Note that the set/get (multiple) values commands will not necessarily provide the most precise error code possible – you might for example get the "REGISTER_WRITE_UNSUCCESSFUL" error even though "REGISTER_IS_READ_ONLY" might be more precise. The precise reasons for a set/get failing can be determined by examining the register table for the relevant system.
