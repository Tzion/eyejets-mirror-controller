MR-E-2  138-161-00
Resonant Amplitude Control for Optical Feedback

Resonant Amplitude Control system ID: 0x35

This is a control loop acting on optical feedback of resonant mirror units.

Register Map:

Address Name Default Range Description Type Access
0x3500 Enable control false or 0 0 - 1 Enable control of resonant axis bool read / write
0x3501 Force res. frequency measurement N/A N/A Reading triggers resonant frequency measurement
(takes cca 200ms, adjust communication timeouts accordingly) 		float read only (A)
0x3502 Resonant (natural) frequency N/A * N/A Returns resonant frequency of the resonant axis based on startup measurement float read only
0x3503 Actual (driven) frequency N/A N/A Actual driving frequency in Hz
(not recommended to change when control is enabled) 		float read / write (B)
0x3504 Actual mechanical amplitude N/A ** N/A Actual driving amplitude (angle) in degrees float read only
0x3505 Desired mechanical amplitude 2.5 0 - 12.5 Desired driving amplitude (angle) in degrees /limited to 12.5°/ float read / write (B)
0x3506 Actual field of view N/A N/A Actual field of view (angle) in degrees float read only
0x3507 Desired field of view 10.0 0 - 50 Desired field of view (angle) in degrees /limited to 50°/ float read / write (B)
0x3508 Current driving resonant axis 0.07 or 0.12 *** 0 - 0.3 Driving current setting in A
(not recommended to change default value; limited to 0.3A) 		float read / write (B)
0x3509 Frequency shift 3.0 0 - 10 Frequency shift for initial control loop (Hz) float read / write (B)
0x350A Frequency delta coefficient 0.5 0.1 - 0.9 Coefficient adjusting step in calculation of control frequency float read / write (B)
0x350B Frequency tolerance 25 0 - 50 Tolerance in % for limiting frequency control range float read / write (B)
0x350C Resonance slope -1.0 -2.0 - -0.1 Slope of amplitude frequency characteristic (deg/Hz) float read / write (B)
0x350D Time window 0.005 0.001 - 0.0256 Integration time for amplitude measurement in seconds float read / write (B)
0x350E Averaging cycles 10 1 - 100 Number of cycles for average measurement of amplitude (at frequency 10kHz)
(averaging timing = Time window * Averaging cycles) 		uint32 read / write (B)
0x350F Control cycles 25 1 - 100 Number of wait cycles before frequency control applies changes
(control timing = Time window * Averaging cycles * Control cycles) 		uint32 read / write (B)

Note 1: If startup measurement fails, system Status register bit# 4 "Mirror not stable" is set
Note 2: If mechanical limit was reached, system Status register bit# 5 "Linear output limit is reached" is set
Note 3: Default driving current is set depending on type of resonant mirror; 0.07A or 0.12A are maximum values, current used for driving is calculated using desired/maximal amplitude ratio. Setting current value after setting mechanical amplitude overrides calculated driving current

Note (A): Register reading will stop and restart resonant amplitude control in case it is running
Note (B): Writing to register when resonant amplitude control is running will fail with error REGISTER_IS_LOCKED (0x0109)
