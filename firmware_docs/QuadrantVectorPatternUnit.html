Quadrant Vector Pattern Unit

Quadrant Vector Pattern Unit system ID base: 0x80

Quadrant vector pattern unit implements quadrant control upon external synchronisation signals SYNC_A and SYNC_B. Validity of signal level sequence is checked by a state machine with following pattern:

Transition Description SYNC_A level SYNC_B level
A Quadrant4 to Quadrant1 0 (low) 0 (low)
B Quadrant1 to Quadrant2 0 (low) 1 (high)
C Quadrant2 to Quadrant3 1 (high) 1 (high)
D Quadrant3 to Quadrant4 1 (high) 0 (low)
