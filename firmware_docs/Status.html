MR-E-2  138-161-00
Status

Status system ID: 0x10

Status system provides information about the firmware identity, version and current status of the driver.

0x1007 System status register 0 - No error Register encodes system status and errors
Read:
Bit#
0 - Proxy not connected
1 - Proxy temperature threshold is reached. See Temperature Manager
2 - Mirror temperature threshold is reached See Temperature Manager
3 - Mirror EEPROM not valid
4 - Mirror not stable. See stability criterion in Optical Feedback
5 - Linear output limit is reached. See Linear Output
6 - Linear output average limit is reached. See Linear Output
7 - XY input is trimmed. See Input Conditioning
8 - Proxy was disconnected
9 - Proxy temperature threshold was reached
10 - Mirror temperature threshold was reached
11 - Linear output limit was reached
12 - Linear output average limit was reached
13 - XY input was trimmed
14..31 - Reserved
Write:
Writing to this register resets all history error flags 8, 9, 10, 11, 12, 13
uint32 read write
0x1008 Proxy FPGA version N/A uint32 read only
0x1009 CPU FPGA version N/A uint32 read only
Note
System status register is updated every 0.5 ms.
