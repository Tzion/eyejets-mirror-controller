MR-E-2  138-161-00
Signal Flow Manager

Signal Flow Manager system ID: 0x40

The Signal Flow Manager is providing registers to set up signal flow systems for each stage. The specific register address depends on the index of signal flow stage and the index of the channel. The general address of the Signal Flow Manager register controlling stage with index S in channel with index C is: 

S + 5*C
Attention
Stages and channels are indexed from 0.

The Signal Flow consists of five stages.

Signal Flow Stage Index
Input stage 0
Input conditioning stage 1
Control stage 2
Output conditioning stage 3
Output stage 4

For example, the register for the control stage (index 2) of the second channel (index 1) is 2 + 5*1 = 7

Register Map:

Address Name Default Value Description Type Access
0x4000 Active input system of channel one 0x50 System ID is expected uint32 read write
0x4002 Active control system of channel one 0xB0 System ID is expected uint32 read write
0x4005 Active input system of channel two 0x51 System ID is expected uint32 read write
0x4007 Active control system of channel two 0xB1 System ID is expected uint32 read write

Examples:

To activate the signal generator system for channel two, set to register with address 0x4005 the value 0x61.

To activate the calibrated PID system of channel one, set to register with address 0x4002 the value 0xC0.
